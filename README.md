# MRU - práctica en Python

Este repositorio contiene un código simple en Python desarrollado como
práctica para afianzar los siguientes conceptos:

- Definición y uso de funciones
- Uso de `return`
- Condicionales `if`, `elif` y `else`
- Cálculo de la posición en MRU
- Cálculo de la distancia entre dos móviles en una dimensión

## Consideraciones físicas

- El movimiento se modela en una sola dimensión (eje x).
- Se asume MRU.
- Se considera únicamente el sentido positivo del eje x.
- No se analizan velocidades negativas.
- El objetivo del código es didáctico, no un modelo físico completo.
